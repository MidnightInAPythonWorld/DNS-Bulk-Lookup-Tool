# DNS-Bulk-Lookup-Tool

#### Purpose
 - Perform bulk DNS A lookups
 - Script is using threading to increase speeds
 - Performs about 5k lookups in 3:30 minutes
 
#### Input
 - Text file with list of domains
 
#### Output
 - A CSV file is generated with results
